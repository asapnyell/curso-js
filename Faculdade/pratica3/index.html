1) Lista (não ordenada) com 10 tags HTML

Aqui estão 10 tags HTML comuns usadas na construção de páginas web:

  • <h1> — Título principal
  • <p> — Parágrafo
  • <a> — Link (âncora)
  • <img> — Imagem
  • <ul> — Lista não ordenada
  • <ol> — Lista ordenada
  • <li> — Item de lista
  • <div> — Contêiner genérico
  • <span> — Conteúdo em linha
  • <form> — Formulário

2) Lista aninhada (ordenada) — Tags divididas por categorias

As tags HTML são categorizadas com base em sua função e uso. Aqui estão algumas categorias comuns:

  1. Estrutura de página
    1. <html>
    2. <head>
    3. <body>
    4. <meta>
  2. Semântica & Conteúdo
    1. <header>
    2. <main>
    3. <footer>
    4. <article>
  3. Formatação de texto
    1. <h1>, <h2>, <h3>
    2. <strong>
    3. <em>
    4. <small>
